# server(nodejs)
## to run the server
```
 cd /server 
 run cmd 'npm start'
```
## setup environments
```
create new file '.env' on directory /server and defined enviroments that used in application
for example add this "ACCESS_TOKEN_SECRET=12345" into '.env' 
```


# client(react)
## to run the client
```
cd /client
```
